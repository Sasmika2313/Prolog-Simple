# Prolog Simple

A simple Prolog-based expert system with a web interface.

## Project Structure
- `main.py`: Flask application.
- `diagnose.pl`: Prolog knowledge base.
- `templates/`: HTML templates.

## Setup
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the application:
   ```bash
   python main.py
   ```
